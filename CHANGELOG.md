# Changelog

All notable changes to Prompto will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-25

### Added

- **Sidebar prompt library** — card-based browsing with search, category filtering, and favorites
- **Context-aware adaptation** — auto-detects active note, selected text, and backlinks
- **Multi-LLM support** — Grok (xAI), Claude (Anthropic), GPT-4o (OpenAI), Venice.ai, and Ollama (local)
- **Visual prompt creator** — full GUI with icon picker, tag chips, and category management
- **Manage Library modal** — sortable table with bulk actions (set category, add tags, set rating, find & replace, delete)
- **Import / Export** — JSON, CSV, Markdown, YAML, and print-ready PDF (HTML)
- **Effectiveness tracking** — 1–5 star ratings on LLM responses, stored in frontmatter
- **Response viewer** — Markdown-rendered responses with copy, insert, and rating
- **Quick settings access** — cog button in sidebar for direct access to API keys and configuration
- **8 sample prompts** included out of the box
